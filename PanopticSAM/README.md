##Installation:
1. Install SAM model of choice(Vit-b/l/h) from https://github.com/facebookresearch/segment-anything
2. pip install requirements.txt
##Usage
SAMM2FORMER.py has training code for cityscapes including dataset and model in one file
SAMM2FORMERCOCO.py has training code for COCO dataset including dataset and model in one file
SAM2FORMER.ipynb has inference code for the model in single file
